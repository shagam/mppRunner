Mpprunner

Basic running speed benchmark for java and c

View on GitHub Download .zip Download .tar.gz

Abstract.

trivial running speed benchmarks written in Java and c Many threads increment one static integer counter

usage

  • java -jar dist/mpprunner.jar
  • java -jar dist/mpprunner.jar [threads=32] [iterations=1000000] [verbose]

for c:

  • make;
  • mppRunner;
  • mppRunner [threads=32] [iterations=1000000] [verbose] [loose] [atomic]
  • When running with no parameters, uses default parameters.

arguments

Unique prefix of argument name is enough (No need to type full name).

  • treads - number of concurrent threads
  • iterations - loops
  • atomic - c only, use hardware lock (faster)
  • loose - c only, no lock, faster (wrong result)

help

When running with no parameters (Or with) help is printed. The help is created automatically. See Args.java args.c
